-> main is inversion of control - for the library developer is using those functions directly so control remains to the developer.
   but in case of framework, framework told us where to write which code so control remains at the framework.

● What is async and defer?
-> Async: while parsing the HTML file by the browser, if script tag with async attribute comes, while parsing only it will load that script, then resume the parsing
and start executing that script, after execution it will resume the parsing of HTML.
-> defer: it will load all the scripts while parsing only. After HTML parsing completed then only it starts executiion of scripts one by one.
